Install Guide

Android 설치 가이드

APK 파일 설치가 처음이신가요? 아래 단계를 따라하세요.

1

APK 다운로드 — 위 버튼을 눌러 APK 파일을 다운로드합니다.

2

출처 허용 — 설정 → 보안 → "알 수 없는 출처의 앱 설치"를 허용합니다. (최초 1회)

3

설치 실행 — 다운로드된 APK 파일을 탭하여 설치를 진행합니다.

4

앱 실행 — 설치 완료 후 앱을 열고 회원가입을 진행하세요. 바로 게임을 시작할 수 있습니다!
